Ce dossier centralise l’ensemble des paramètres de configuration du projet.

Il permet de découpler la logique métier du code des paramètres expérimentaux
(taux d’erreur, activation des attaques, choix cryptographiques, scénarios de simulation).

Cette approche garantit :
- la reproductibilité des expériences,
- la comparabilité des résultats,
- une modification rapide des scénarios sans impact sur le code source.
