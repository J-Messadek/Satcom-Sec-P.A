Ce dossier contient l’ensemble du code source du projet.

L’architecture est découpée par briques fonctionnelles indépendantes,
reflétant une chaîne de communication satellite réaliste.

Cette séparation permet :
- un développement parallèle par les membres de l’équipe,
- une lisibilité accrue,
- une analyse précise de l’impact de chaque mécanisme.
