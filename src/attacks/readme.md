Ce dossier contient les scénarios d’attaques simulées.

Les attaques sont volontairement isolées du reste du code afin de :
- pouvoir être activées ou désactivées facilement,
- analyser leur impact de manière contrôlée,
- étudier l’efficacité des mécanismes de défense associés.

Chaque attaque cible une étape précise de la chaîne.
