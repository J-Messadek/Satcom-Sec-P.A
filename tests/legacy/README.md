# Brouillons conservés (historique)

Ce dossier rassemble d'anciens scripts exploratoires, conservés à titre
d'historique. Ils ne sont **pas** collectés par pytest (cf. `norecursedirs`
dans `pyproject.toml`) et ne font pas partie de la suite de tests active.

- `creation_tram2.py`  : première exploration de la création de trames CCSDS.
- `reading_image.py`   : lecture d'une image en blocs d'octets.
- `dashboard_hello.py` : maquette initiale du dashboard Streamlit.
