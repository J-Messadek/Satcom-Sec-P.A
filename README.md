# satcom-sec

Ce projet vise à modéliser, attaquer et sécuriser une chaîne de communication satellite simulée,
depuis une transmission volontairement vulnérable jusqu’à une communication résiliente face au bruit
et aux attaques cyber.

L’architecture est volontairement modulaire afin de :
- séparer clairement les responsabilités techniques,
- faciliter le travail collaboratif à trois,
- permettre une évolution progressive du projet (attaques, défenses, métriques).

Chaque dossier correspond à une brique fonctionnelle précise du système.
